# JavaScript Training

```
$ git clone https://github.com/ufotsuboi/training-javascript-curry.git
$ cd path/to/repo
$ npm i
$ npm test
```

- `1.curriedSum/curriedSum.js` を実装してください。
- `$ npm test -- 1.curriedSum` が通れば OK です。
